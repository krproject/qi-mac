# qi-mac

器（qi），工欲善其事，必先利其器！  

本工程包含了在Mac上进行krproject开发所需的工具及软件依赖，方便以命令行的方式一键安装！

# 安装使用

1. 安装Homebrew   
  参考：https://brew.sh

2. 添加Tap  
```bash
brew tap krproject/qi https://github.com/krproject/qi-mac.git
```

3. 安装工具软件  
```cmd
brew install krproject/qi/sts
brew install krproject/qi/vscode
...
```

4. 配置工具软件  
  。。。
